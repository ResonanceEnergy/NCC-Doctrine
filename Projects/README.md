# Projects Directory

## Purpose
Active and archived enterprise projects, initiatives, and special programs across the NCC organization.

## Ownership
- **Primary Owner**: NCC Project Management Office (PMO)
- **Secondary Owner**: Department Leads (project-specific)
- **Maintenance Lead**: Project Operations Team

## Directory Structure
- `/projects/` - Main projects directory
  - `active/` - Currently active projects and initiatives
  - `completed/` - Successfully completed projects (archived)
  - `on-hold/` - Projects temporarily suspended
  - `cancelled/` - Cancelled projects and post-mortems

## Project Categories
- **MicroFlowHydro**: Global product transformation and implementation
- **UPI**: Uruguay-Paraguay Integration project
- **Augmented Arbitrage**: Financial trading and arbitrage systems
- **Biotech**: Biotechnology research and development
- **Quantum Computing**: Quantum technology initiatives

## Maintenance Procedures
- **Project Lifecycle**: All projects follow standardized PMO lifecycle methodology
- **Status Reporting**: Weekly status reports for active projects
- **Resource Allocation**: Quarterly resource planning and budget reviews
- **Risk Management**: Project risk assessments conducted monthly
- **Documentation**: Complete project documentation required for all phases
