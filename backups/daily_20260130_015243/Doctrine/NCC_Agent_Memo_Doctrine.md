## DOCTRINE STATEMENT

**EVERY AGENT WILL OUTPUT A MEMO EVERY CYCLE THAT CONTAINS ALL THE INFORMATION THE AGENT PRODUCED IN CONTEXT WITH DESCRIPTION AND WILL BE SAVED AS A TEXT FILE AND EMAILED TO THE AX DEPARTMENT.**

This doctrine is now INSTITUTED as core NCC operational protocol and supersedes all previous agent reporting requirements.

================================================================================
DOCTRINE REQUIREMENTS - MANDATORY COMPLIANCE
================================================================================

### 1. CYCLE OUTPUT MANDATE
- **MANDATORY**: Every agent must generate a comprehensive memo at the completion of each operational cycle
- **AUTOMATION**: Memo generation is integrated into all agent task tracking systems
- **TRIGGER**: Automatic execution following Daily Check, Cross-Department Sync, and major operational cycles

### 2. MEMO CONTENT REQUIREMENTS
Every memo MUST contain the following sections with complete and accurate information:

#### AGENT IDENTIFICATION
- Agent ID (unique identifier)
- Cycle ID (timestamp-based cycle reference)
- Division/Department affiliation
- Subsidiary company
- Security clearance level
- AZ Rank and AX Optimization Level
- Cycle start/end timestamps

#### CYCLE SUMMARY
- Overall cycle status (Successful/Failed/Partial)
- Cycle duration and efficiency metrics
- System uptime and performance indicators

#### TASKS EXECUTED
- Complete list of all tasks performed during the cycle
- Task status (Completed/In Progress/Failed)
- Execution priority levels
- Time spent on each task
- Detailed task descriptions and outcomes

#### RESULTS ACHIEVED
- Quantifiable results and achievements
- Impact assessments (High/Medium/Low)
- Performance metrics and KPIs
- Data processing statistics
- Efficiency improvements documented

#### DATA PROCESSED
- Total data points processed
- Data processing rates
- Data sources accessed
- Data quality scores
- Retention compliance status

#### COMMUNICATIONS SENT
- All internal/external communications
- Message types and recipients
- Communication priorities
- Response requirements and timelines

#### ISSUES ENCOUNTERED
- Technical issues and resolutions
- Operational challenges
- Security incidents (if any)
- Performance bottlenecks identified

#### RECOMMENDATIONS
- Process improvements suggested
- Resource optimization opportunities
- Training requirements identified
- System enhancement recommendations

#### NEXT CYCLE PLAN
- Planned tasks and priorities
- Resource requirements
- Risk mitigation strategies
- Performance targets

#### PERFORMANCE METRICS
- Efficiency rating (target: 99.9%)
- AZ Compliance score (target: 100%)
- AX Optimization score (target: 95%)
- Cross-departmental coordination rating
- System uptime percentage
- Error rates and incident counts

### 3. FILE FORMAT AND NAMING
- **Format**: Plain text (.txt) file
- **Encoding**: UTF-8
- **Naming Convention**: NCC_AGENT_MEMO_{AgentID}_{CycleID}_{Timestamp}.txt
- **Storage**: Local agent_memos directory with automatic archiving

### 4. DISTRIBUTION PROTOCOL
- **Primary Recipient**: AX Department Intelligence Analysis
- **Distribution Method**: Automated email system
- **Email Subject**: NCC AGENT MEMO DOCTRINE - {AgentID} Cycle {CycleID}
- **Delivery Confirmation**: Required and logged
- **Backup Distribution**: Secondary distribution to AZ PRIME oversight

### 5. COMPLIANCE MONITORING
- **Monitoring System**: Automated compliance tracking
- **Reporting Frequency**: Real-time monitoring with daily reports
- **Non-Compliance Actions**: Automatic escalation to AZ PRIME Command
- **Compliance Targets**: 100% adherence required
- **Audit Trail**: Complete distribution and receipt logging

2. Task tracker automatically triggers memo generation
3. Memo doctrine system collects all cycle data
4. Comprehensive memo is generated with all required sections
5. Memo is saved as text file with standardized naming
6. Automated email distribution to AX Department
7. Compliance logging and monitoring updates
8. Archive process removes memos older than retention policy
